# 🎯 Invigilator Allocation Management System

A Django-based web application to automate the allocation of invigilation duties for faculty during examinations.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 👨‍🏫 Faculty Dashboard (View assigned duties)
- 🧑‍💼 HOD Dashboard (Analytics & Reports)
- ⚙️ Automatic Invigilator Allocation
- 📅 Session-based allocation (Forenoon / Afternoon)
- 📄 PDF Report Generation (Faculty & Full Report)
- 🚫 Absence Handling (Avoid assigning unavailable staff)

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- PDF Generation: ReportLab

---

## ⚡ How It Works

1. Admin creates exam schedule (date + session)
2. Marks absent lecturers
3. System auto-allocates invigilators to rooms
4. Faculty can view & download their duties
5. HOD can generate full allocation reports

---

## 📌 Key Highlights

- Ensures **no duplicate session allocation**
- Balanced duty distribution
- Clean dashboard UI
- Real-time allocation results
